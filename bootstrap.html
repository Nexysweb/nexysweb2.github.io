Difference between HTML and Bootstrap

  • Bootstrap is an HTML framework
  • Developed by engineers at Twitter
  • Standardized compenents
  • Leader (used to be?) in mobile web
  • Most famous HTML framework

Putting everything together

Technology Description Framework/Product of choice
HTML(5) Structure content in different tags semi-structured content Bootstrap
CSS - Cascading Stylesheet arranges the different tags in a beautiful and esthetic way Bootstrap
Javascript Allows interaction between HTML elements and with the server through user interaction and/or external inputs in realtime Mostly angularJS, but sometimes JQuery
Git Revision System - allows to go back in time and keep track of progress as well as optmising colaboration with collaborators GitHub (to be preferred, see below) and also Bitbucket

Install and use Bootstrap

Probably the best way and safest way to kick off a project using bootstrap is to start-off with a bootstrap and include dependencies from a bower repository

Install with bower

If bower is not installed please install bower. Bower is a package manager for the web. bower install bootstrap
